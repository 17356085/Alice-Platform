## 页面职责
岗位的 CRUD 管理。支持岗位名称、编码、所属部门的维护。

## 核心元素（从代码推断）
- 搜索区：岗位名称/编码输入 + 部门筛选 + 搜索/重置按钮
- 表格区：岗位列表（编码/名称/部门/编制人数/状态）
- 操作按钮：新增一行 / 行内编辑 / 删除
- 弹窗：新增/编辑表单（名称/编码/部门/编制人数/描述）
- 分页区：分页器

## 关键交互
- 新增岗位→行内新增或弹窗→提交→表格刷新
- 编辑→行内编辑→保存→表格刷新
- 删除→确认弹窗→删除→表格刷新
- 搜索→表格异步刷新

## 权限与角色
- 可见角色：admin、部门主管
- 可操作角色：admin（全部）、部门主管（本部门）

- 前端校验：编码唯一性、名称必填、编制人数≥0
