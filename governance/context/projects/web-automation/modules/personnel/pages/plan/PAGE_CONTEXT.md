## 页面职责

## 核心元素（从代码推断）
- 搜索区：计划名称/编号输入 + 状态下拉 + 日期范围 + 搜索/重置
- 表格区：计划列表（编号/名称/培训类型/开始日期/结束日期/状态/操作）
- 操作按钮：新增 / 编辑 / 删除 / 发布 / 查看详情
- 弹窗（多步骤）：新增/编辑表单（基本信息→关联课程→指定培训对象→确认）
- 分页区：分页器

## 关键交互

- 发布计划→状态变更→通知培训对象
- 编辑→弹窗→修改→保存
- 状态流转：草稿→已发布→进行中→已完成
