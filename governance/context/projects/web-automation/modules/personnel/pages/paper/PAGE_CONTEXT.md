# PAGE_CONTEXT — personnel / paper

## 基本信息
- 页面ID：personnel-paper
- 页面名称：试卷管理
- 所属模块：人员管理（personnel）→ 培训管理
- 路由：`#/personnel/training/paper`
- 自动化代码：`page/personnel_page/PaperManagePage.py` + `script/personnel/test_paper_management.py`
- 代码规模：1531行（本模块第二大PageObject）

## 页面职责
试卷的完整管理。支持手动选题组卷、自动随机组卷、试卷预览、发布。

## 核心元素（从代码推断）
- 搜索区：试卷名称/编号输入 + 状态下拉 + 搜索/重置
- 表格区：试卷列表（编号/名称/总分/题数/状态/创建时间/操作）
- 操作按钮：新增 / 编辑 / 删除 / 发布 / 预览
- 弹窗（组卷）：新增/编辑表单（基本信息→手工选题→自动抽题规则→预览确认）
- 分页区：分页器

## 关键交互
- 新增试卷→弹窗→填基本信息→手工选题（按题型/分类筛选→勾选题目）→确认→表格刷新
- 自动组卷→设置规则（题型分布/难度分布/每题分值）→系统随机抽题→预览
- 发布试卷→状态变更→关联考试可用

## 权限与角色
- 可见角色：admin、培训管理员
- 可操作角色：admin、培训管理员
- 特殊限制：已发布的试卷不可修改题目

## 特殊行为（待确认）
- 双模式组卷：手工选题 + 自动抽题
- 自动抽题规则：按题型比例、难度分布、分类范围随机抽取
- 选题区：左(题库列表) 右(已选题目) 双栏布局
- 前端校验：总分>0、至少1题、每题分值>0

## 依赖
- 接口：试卷 CRUD + 自动抽题接口
- 上下游页面：题库管理（question，题目来源）、考试管理（exam，关联试卷）
