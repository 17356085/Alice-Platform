# PAGE_CONTEXT — personnel / question

## 基本信息
- 页面ID：personnel-question
- 页面名称：题库管理
- 所属模块：人员管理（personnel）→ 培训管理
- 路由：`#/personnel/training/question`
- 自动化代码：`page/personnel_page/QuestionBankPage.py` + `script/personnel/test_question_bank.py`

## 页面职责
题库的 CRUD 管理。支持多种题型（单选/多选/判断/填空/简答），题目分类和难度标注。

## 核心元素（从代码推断）
- 搜索区：题目内容输入 + 题型下拉 + 分类下拉 + 难度下拉 + 搜索/重置
- 表格区：题目列表（编号/题干摘要/题型/分类/难度/创建时间/操作）
- 操作按钮：新增 / 编辑 / 删除 / 批量导入
- 弹窗：新增/编辑表单（题型/分类/难度/题干/选项A-D/正确答案/解析）
- 分页区：分页器

## 关键交互
- 新增题目→弹窗→选择题型→填写题干+选项+答案→提交→表格刷新
- 选择题型后动态调整表单结构（单选题显示4个选项，判断题显示对/错，简答题显示文本框）
- 搜索→异步刷新

## 权限与角色
- 可见角色：admin、培训管理员
- 可操作角色：admin、培训管理员

## 特殊行为（待确认）
- 动态表单：题型切换时，选项区和答案区结构变化
- 前端校验：题干必填、正确答案必选/必填、选项至少2个
- 批量导入：Excel 上传解析

## 依赖
- 接口：题目 CRUD + 批量导入接口
- 上下游页面：试卷管理（paper，从题库抽题）
