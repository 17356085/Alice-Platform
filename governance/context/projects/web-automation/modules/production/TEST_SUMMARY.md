## 关键发现

### 1. 按钮点击策略
- **Native click 频繁被拦截** — Element Plus 遮罩/动画导致
- **JS click 方案稳定** — `_js_click_by_text()` 绕过所有拦截
- 建议：对弹窗触发类按钮统一使用 JS click

### 2. 测试隔离问题
- **module 级 driver_setup + function 级 page** 导致弹窗残留
- 解决方案：fixture teardown 中 JS 关闭所有弹窗 + Escape 兜底
- 全量串行运行时仍有 ~20% 概率不稳定，建议分批次独立运行

### 3. 页面定位经验
- **section-title** 元素在 dashboard 区是 `<span>`，生产区是 `<div>` — 用 `//*` 通配
- **趋势分析弹窗无取消按钮** — 需用 `×` (`.el-dialog__headerbtn`) 关闭
- **日期选择器** 是页面加载完成的可靠信号
