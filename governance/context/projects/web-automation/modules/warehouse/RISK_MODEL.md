## 高风险路径

1. **审批全链路**：新增入库 → 选择危废品 → 提交申请 → 主管审批(chenqian) → 仓管审批 → 审核完成 — 任一步失败则业务中断
2. **库存闭环**：入库→库存增加→出库→库存减少→盘点核对 — 任一步数据错误导致库存失真
3. **子弹窗交互**：弹窗A（新增入库）→ 弹窗B（选择危废品）→ 回显 → 提交 — 弹窗嵌套的稳定性
4. **角色权限边界**：备品管理员访问环保页面、仓库领用员越权操作 — 横向越权
