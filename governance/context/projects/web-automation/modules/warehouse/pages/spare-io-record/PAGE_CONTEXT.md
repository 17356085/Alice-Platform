# 备品出入库记录 — 页面上下文

## 页面信息
- 路由: 从左侧导航 `库管管理` → `备品备件管理` → `出入库记录`
- 路由 Hash: `#/warehouse/spare/io_record`（从 conftest.py 推断）
- Page Object: `SpareIORecordPage`（位于 `page/warehouse_page/SpareIORecordPage.py`）
- 测试脚本: `test_spare_io_record.py`（位于 `script/warehouse/test_spare_io_record.py`）
- 页面类型: 只读审计日志页面，无审批流，无增删改
- 权限类型: 只读（仅可查询和重置）

## 页面整体结构
- **搜索区**: 位于页面顶部，包含物品名称输入框、查询/重置按钮
- **主内容区**: 展示 `el-table` 表格，列出所有备品出入库操作记录
- **分页区**: 位于表格下方，包含 `el-pagination` 分页器
