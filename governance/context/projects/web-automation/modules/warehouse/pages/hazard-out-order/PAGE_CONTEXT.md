## 页面职责

- 展示环保危废出库单据列表，支持按经办人/日期/流程状态筛选
- 提供新增出库单功能（含危废品选择器子界面）
- 出库单提交后走审批流程（→ 工作流管理，与入库共用同一审批模板）
- 支持查看出库单详情、编辑可修改状态的单据
