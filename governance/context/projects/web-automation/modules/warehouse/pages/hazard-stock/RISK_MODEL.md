## 业务场景

### 角色与旅程

| 角色 | 职责边界 | 典型操作 |
|---|---|---|
| 库管员 | 日常查看危废品库存数量，确认库存状态 | 查看、搜索 |
| 环保专员 | 关注危废品库存总量和合规性 | 查看、搜索 |
| 系统管理员 | 配置权限，监控系统使用 | 查看 |

### 典型业务旅程

**库管员 - "查询某危废品库存" 旅程**
1. **进入**: 导航到 `库管管理 > 环保危废管理 > 库存查询`，页面加载并展示全量库存列表。
2. **搜索**: 在搜索框输入危废品名称（如"废酸"），点击"查询"按钮。
3. **查看**: 表格筛选显示匹配的危废品库存记录，包含名称、数量、仓库等信息。
4. **重置**: 点击"重置"按钮，列表恢复为全量库存数据。旅程结束。

### 业务流程

- **Happy Path**: 库管员查看库存 → 搜索特定危废品 → 查看库存数量 → 重置搜索。
- **空数据 Path**: 库管员搜索不存在的危废品 → 系统显示空数据提示。
- **Exception Path**: 查询接口异常 → 系统显示错误提示。

### 业务规则

| ID | 规则描述 | 触发点 | 可验证性 |
|---|---|---|---|
