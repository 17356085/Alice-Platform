## 审批流

```
提交申请 → 主管审批(admin/chenqian 会签) → 库管确认(tjw)
```

---

## 关键交互

- 新增 → 弹窗 → 填写申请信息+选择备件 → 保存草稿
- 提交 → 触发审批链 → 状态变为"待审核"
- 删除 → 仅草稿/已退回可删
- 关联出库：审批通过后 → 备品出库页面生成出库单（推测）

---

## 与备品出库的关联

备品出库页面有"关联申请单"列，LY单号链接指向领用申请单。领用申请通过后驱动出库流程。
