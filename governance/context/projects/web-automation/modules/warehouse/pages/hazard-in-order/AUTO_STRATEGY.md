## 4. 等待策略建议

该页面存在多种异步行为，建议的等待策略如下：

-   **页面加载后**: 使用 `wait_vue_stable()` + `_wait_loading_gone()`，这是正确的做法，能覆盖大部分Vue渲染和数据加载场景。
-   **搜索后**: 使用 `wait_vue_stable()`，等待表格数据刷新。建议增加一个“搜索完成后，检查表格第一行数据变化”的显式等待。
-   **点击按钮打开弹窗**: 使用 `wait_dialog_open()` (断言 A级定位器可见)，等待弹窗动画完成。
-   **点击“选择危废品”打开嵌套弹窗**: 这是最复杂的异步点。建议封装备用选项:
    -   **首选**: 等待一个新的 `el-dialog` 或 `el-drawer` 元素变为可见。
    -   **兜底**: 轮询检查 `el-overlay` 的数量，直到出现新的遮罩层。
    -   **超时**: 所有这些等待都应设置一个超时时间（如5-8秒），超时后抛出明确的异常。
