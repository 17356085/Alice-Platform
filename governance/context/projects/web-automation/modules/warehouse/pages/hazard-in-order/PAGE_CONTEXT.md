## 关键交互

### 新增入库完整流程

```
① 点击"新增入库" → 弹窗A展开（920px）
② 填写"入库时间"（日期选择器）
③ 填写"经办人"（文本输入）
④ 点击"选择危废品" → 弹窗B展开
    ├── 选择"危废品分类"（下拉）
    ├── 输入"危废品名称"（搜索）
    ├── 点击"重置"可清空筛选
    └── 勾选危废品行 → 确认 → 弹窗B关闭 → 已选危废品回显在弹窗A中
⑤ 点击"提交申请" → 弹窗A关闭 → 表格刷新 → 新单据（状态=待审核）
```

### 查询流程

```
选择 流程状态（下拉） + 输入 经办人 + 选择 日期 → 点击"查询" → 表格异步刷新
点击"重置" → 清空所有筛选条件 → 表格恢复全量
```

### 查看流程

```
点击行内"查看" → 弹窗C展开（只读）
  ├── 查看基本信息（单号/时间/经办人/审核人）
  ├── 查看危废品明细列表
  └── 查看流程跟踪时间线
```

### 编辑流程（推测）

```
仅当状态为"待提交"或"已退回"时，行内出现"编辑"按钮
点击"编辑" → 弹窗（类似新增，但预填已有数据）→ 修改 → 重新提交
```

实测 3 步流程：

```
① 提交申请 (admin)        ──→ 🟢 自动完成
② 主管审批 (chenqian)      ──→ 🟢 已完成
③ 仓管审批                  ──→ ⬜ 待处理
```

| 确认事项 | 结论 |
|----------|------|
| 审批流来源 | ✅ 系统预配的**危废出库审批链** |
