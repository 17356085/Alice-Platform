# Web Modules

## 目录职责
本目录用于承接 Web 自动化项目的模块级上下文。

## 预期结构
- modules/<module>/MODULE_CONTEXT.md
- modules/<module>/pages/<page>/PAGE_CONTEXT.md
- modules/<module>/pages/<page>/RISK_MODEL.md
- modules/<module>/pages/<page>/TEST_DESIGN.md
- modules/<module>/pages/<page>/TEST_CASES.md
- modules/<module>/pages/<page>/TECH_ANALYSIS.md
- modules/<module>/pages/<page>/AUTO_STRATEGY.md

## 当前策略
当前优先保持与存量 contexts/ 一致的模块划分，不强制复制已有内容。
新增模块或新增页面，优先按这里的规范落位。
