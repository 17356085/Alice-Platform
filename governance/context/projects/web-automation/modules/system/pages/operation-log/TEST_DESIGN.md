# TEST_DESIGN — system / operation-log

## 基本信息
- 模块：system
- 页面：操作日志
- 测试目标：验证页面加载、多条件搜索、详情查看、清空（只读审计页面）
- 需求来源：PAGE_CONTEXT.md + RISK_MODEL.md

## 测试范围
- 包含：页面加载、搜索(模块/类型/操作人/状态/时间)、详情弹窗、清空、分页
- 不包含：敏感信息脱敏验证（手工安全测试）

## 数据策略
- 测试数据来源：依赖系统已有操作记录（无需造数据）
- 数据清理方式：不清理（只读页面）
- 前置准备：admin账号登录

## 自动化建议
- 建议自动化：页面加载/多条件搜索/详情/分页 — 定位器稳定
- 不建议自动化：敏感字段脱敏验证（需人工判断）
