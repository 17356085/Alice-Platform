## 3. 搜索/筛选区

### 3.1 元素清单（实测）

| 元素ID | 标签文字 | 控件类型 | placeholder/默认值 | 真实 DOM 特征 |
|:---:|------|:---:|------|------|
| **SRC-01** | 搜索 | `<el-input>` (suffix, width:240px) | `搜索用户名/姓名/手机号` | `input.el-input__inner` |
| **SRC-02** | 角色 | `<el-select>` (width:200px) | 默认: `全部角色` | `div.el-select` + Teleport dropdown |
| **SRC-03** | 状态 | `<el-select>` (width:120px) | 默认: `全部` | `div.el-select`（⚠️ **不是** radio-group） |

### 3.2 ⚠️ 与 Page Object 差异

| 项目 | PO 假设 | 实测 | 影响 |
|------|---------|------|------|

## 5. 数据表格

### 5.1 列定义（实测）

| 列序号 | 列标题 | 宽度 | 数据类型 | 渲染方式 | 示例数据 |
|:---:|------|:---:|:---:|------|------|

### 5.2 ⚠️ 列标题与 PO 差异

| PO 假设 | 实测 | 说明 |
|------|------|------|
| 部门 | **组织名称** | 列标题不是"部门" |
| 创建时间 | **最后登录** | 列标题不是"创建时间"！显示的是用户最后一次登录时间，未登录过显示 `-` |
| 角色列 | 角色 (el-tag) | PO 用 `get_column_data()` 获取文本，但角色列是多个 tag，换行分隔 |

### 5.3 行操作按钮（实测每行 4 个按钮）

| 序号 | 按钮文字 | 位置 | 定位线索 |
|:---:|------|------|------|
| **ACT-01** | **查看** | 第1个 | `td[8]//button[1]` |
| **ACT-02** | **编辑** | 第2个 | `td[8]//button[2]` |
| **ACT-03** | **分配角色** | 第3个 | `td[8]//button[3]` |
| **ACT-04** | **更多** | 第4个 (el-dropdown) | `td[8]//div.el-dropdown/button` |

> ⚠️ **PO 差异**: PO 中 `_get_action_button_by_username` 仅映射 button[1]=编辑、button[2]=分配角色，**未覆盖「查看」按钮**。测试中也从未使用过「查看」功能。

### 5.4 「更多」下拉菜单（实测）

| 菜单项 | 说明 |
|------|------|
| **启用/重置密码** | 合并文案（停用用户显示"启用"，启用用户显示"重置密码"） |
| **删除** | 弹出 MessageBox 确认后删除 |

> ⚠️ **PO 差异**: PO 中 `MORE_RESET_PWD_OPTION` 期望匹配 "重置密码" **或** "启用/重置密码"，实测验证了后者。

## 9. 页面状态

| 状态 | 触发条件 | 实测表现 |
|------|---------|------|
| **加载中（初始）** | 页面首次打开 | `#app` 内蓝色旋转 CSS loader（`.loader`） |
| **加载中（组件）** | 表格数据请求中 | `el-loading-mask` 覆盖表格 |
| **正常（有数据）** | API 返回 200 | 表格渲染 10 行/页，分页显示「共 151 条」 |
| **空数据（搜索无结果）** | 搜索不存在的关键字 | 表格 0 行，分页 0 条（推断） |
| **接口异常** | API 4xx/5xx | Toast 错误提示（推断） |
| **弹窗打开** | 点击新增/编辑/分配角色 | 遮罩 + 600px Dialog |
| **删除按钮 disabled** | 无勾选行 | `el-button--danger.is-disabled` |
| **分页边界** | 第 1 页 / 最后页 | 上一页/下一页按钮 disabled |

## 11. 实测与 PO 代码差异总结

## 12. 自动化定位速查

| 场景 | 推荐策略 | 定位值 |
|------|------|------|
| 搜索输入框 | placeholder 定位 | `//input[@placeholder='搜索用户名/姓名/手机号']` |
| 角色筛选 | el-select 展开 | label="角色" 的 el-form-item → el-select |
| 表格行 | CSS class | `tr.el-table__row` |
| 用户名列 | td:nth-child(2) | `.el-table__body-wrapper td:nth-child(2) .cell` |
| 行操作按钮 | 文本定位 | `//tr[.//div[text()='{username}']]//button//span[text()='编辑']` |
| 新增弹窗 | 表单定位 | `.el-dialog .el-form-item` → label 文本匹配 |
| 部门选择 | tree-select | label="部门" 的 form-item → el-select (tree) |
| Toast 消息 | CSS class | `.el-message__content`（取 [last()]） |
| 分页总数 | CSS class | `.el-pagination__total` |
