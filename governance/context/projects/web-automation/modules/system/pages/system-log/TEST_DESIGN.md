# TEST_DESIGN — system / system-log

## 基本信息
- 模块：system
- 页面：系统日志
- 测试目标：验证页面加载、日志类型/级别筛选、搜索、清空（只读运维页面）
- 需求来源：PAGE_CONTEXT.md + RISK_MODEL.md

## 测试范围
- 包含：页面加载、日志类型下拉筛选、日志级别下拉筛选、模块名称搜索、清空、分页
- 不包含：日志内容语义分析、日志聚合统计

| 场景ID | 测试维度 | 场景描述 | 优先级 | 执行方式 |
|--------|----------|----------|--------|----------|
| TD-SYSLOG-001 | 页面加载 | 系统日志页正常加载 | P0 | 自动化 |
| TD-SYSLOG-010 | 搜索 | 按日志类型下拉筛选 | P0 | 自动化 |
| TD-SYSLOG-011 | 搜索 | 按日志级别下拉筛选 | P0 | 自动化 |
| TD-SYSLOG-012 | 搜索 | 按模块名称搜索 | P1 | 自动化 |
| TD-SYSLOG-013 | 搜索 | 按时间范围搜索 | P1 | 自动化 |
| TD-SYSLOG-014 | 搜索 | 重置搜索条件 | P1 | 自动化 |
| TD-SYSLOG-030 | 清空 | 清空-确认弹窗-取消 | P0 | 自动化 |
| TD-SYSLOG-031 | 清空 | 清空-确认弹窗-确定(⚠️高危仅验证弹窗) | P0 | 自动化 |

## 数据策略
- 测试数据来源：依赖系统已有运行日志（无需造数据）
- 数据清理方式：不清理（只读页面）
- 前置准备：admin账号登录

## 自动化建议
- 建议自动化：页面加载/下拉筛选/搜索/分页 — 定位器稳定
- 不建议自动化：大数据量性能（需独立性能测试环境）
