# TEST_DESIGN — system / org-management

## 基本信息
- 模块：system
- 页面：组织管理
- 测试目标：验证部门树CRUD全流程、搜索筛选、权限控制
- 需求来源：PAGE_CONTEXT.md + RISK_MODEL.md

## 测试范围
- 包含：页面加载、树展开/折叠、搜索筛选、新增/编辑/删除部门、新增子节点、导出
- 不包含：跨浏览器兼容性测试、移动端适配

## 数据策略
- 测试数据来源：运行时生成唯一名称(时间戳后缀)
- 数据清理方式：fixture teardown中按名称前缀删除
- 前置准备：admin账号登录

## 自动化建议
- 建议自动化：搜索/CRUD/树操作 — 定位器稳定
- 不建议自动化：网络异常模拟
- 原因：Selenium无法控制网络层
