# PAGE_CONTEXT — system / org-management

## 基本信息
- 页面ID：org-management
- 页面名称：组织管理（部门管理）
- 所属模块：系统管理（system）
- 页面入口：左侧菜单 → 系统管理 → 组织管理
- 路由：#/system/dept

## 页面职责
- 展示组织架构树形列表，支持树节点展开/折叠
- 支持按组织名称、组织类型、状态搜索
- 提供部门的新增、编辑、删除操作（树形CRUD）
- 支持导出功能

## 关键交互
- 树节点展开/折叠 → 异步加载子节点
- 新增按钮 → 弹窗 → 填写表单(名称/类型/上级/排序) → 确认 → 树刷新 + toast
- 行内"新增" → 弹窗(上级自动填充为当前节点) → 添加子部门
- 删除 → 确认弹窗 → 确认 → 节点移除 + toast（有子节点时后端拦截）
- 搜索 → 表格刷新（树形展平搜索匹配节点）

## 权限与角色
- 可见角色：admin、系统管理员
- 可操作角色：admin(全部)、系统管理员(新增/编辑，不可删除)
- 特殊限制：普通用户不可见此页面

## 特殊行为
- 异步加载：树节点懒加载子节点；搜索后树刷新
- 动态渲染：状态列 el-tag 颜色；有子部门时删除按钮禁用
- 前端校验：名称必填≤50字符；排序号正整数
- 后端校验：名称唯一性；父节点存在性；有子节点禁止删除

## 依赖
- 接口：GET/POST/PUT/DELETE /api/system/dept/*
- 上下游页面：上游-用户管理(用户关联部门)
