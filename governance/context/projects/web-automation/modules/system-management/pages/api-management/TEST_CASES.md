## 自动化优先级建议

- **✅ 强烈建议自动化（P0 & P1 核心流程）**:
  - 页面正常加载、搜索、新增、编辑、删除等核心CRUD操作。
  - 必填项校验、重复数据冲突校验等表单验证。
  - 权限校验（UI层面的按钮隐藏/禁用）。
- **🔶 可以考虑自动化（P2 次要场景）**:
  - 特殊字符搜索、取消操作、并发编辑冲突（较难模拟，但可做）。
- **❌ 不建议自动化（安全/异常场景）**:
  - **权限绕过**、**SQL注入/XSS** 等安全测试：建议使用专门的自动化安全扫描工具（如Burp Suite）或交由人工安全测试团队完成。
