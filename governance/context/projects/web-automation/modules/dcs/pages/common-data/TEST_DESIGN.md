# TEST_DESIGN — dcs / common-data

## 基本信息
- 模块：DCS 数据监控（dcs）
- 页面：常用点位（common-data）
- 测试目标：覆盖常用点位卡片/表格的搜索、右键菜单、拖拽排序
- 需求来源：PAGE_CONTEXT.md + RISK_MODEL.md + CommonDataPage.py

## 测试范围
- 包含：页面加载、搜索/重置、卡片点击、右键菜单、拖拽排序、删除确认
- 不包含：新增（页面为只读查询页）

## 数据策略
- 测试数据来源：使用系统已有的常用点位数据
- 数据清理方式：删除操作后需恢复（恢复默认按钮）
- 前置准备：确保 admin 账号有完整权限

## 自动化建议
- 建议自动化：搜索/重置/卡片读取 — 定位器较稳定
- 谨慎自动化：右键菜单/拖拽排序 — 需要 JS 模拟，定位器待 E2E 确认
- 不建议自动化：恢复默认（不可逆）

- 卡片定位器需根据实际 DOM 调整（test_004 已 skip）
