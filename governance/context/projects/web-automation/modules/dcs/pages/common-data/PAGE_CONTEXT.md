# PAGE_CONTEXT — DCS 常用点位（Common-Data）

## 基本信息
- **页面名称**: 常用点位
- **路由**: `#/common-data`
- **模块**: dcs
- **页面 ID**: dcs-common-data

## 场景描述
常用点位快捷管理页面。显示用户经常关注的参数（温度、压力、流量等）的集合，支持快速添加/删除、自定义排序。

## 页面结构概览

| 区域 | 功能 | 组件类型 |
|-----|------|---------|
| **顶部工具栏** | 搜索、按设备筛选、新增常用点位、导出 | Input + Select + Button |
| **卡片网格** | 常用点位卡片（拖拽排序） | Card Grid + Draggable |
| **操作菜单** | 编辑、删除、移至末尾 | Context Menu / Inline Button |
| **侧栏** | 点位详情面板（可选） | Panel |

## 主要交互元素

### 搜索/筛选区
- 搜索框：按点位名称搜索
- 设备筛选：选择特定设备查看其常用点位
- 刷新按钮：重新加载常用点位列表

### 操作按钮
- 新增常用点位：从全部点位中选择添加（弹窗 or 多选）
- 删除所有：清空所有常用点位（需确认）
- 导出：导出常用点位列表
- 恢复默认：恢复系统预设的常用点位

### 点位卡片
- 显示：点位名称、当前值、单位、最后更新时间
- 状态徽章：正常/故障/离线
- 趋势箭头：上升 ↑ / 下降 ↓ / 平稳 → （基于前一个采集周期）
- 拖拽句柄：允许用户自定义排序

### 卡片操作
- 点击卡片：展开详情（可选）
- 右键/长按：弹出上下文菜单
  - 编辑
  - 删除
  - 置顶
  - 移至末尾

## 已知复杂性
- 支持拖拽排序（Vue Draggable）
- 常用点位数量通常 < 10-20，适合卡片网格展示
- 状态和值需实时刷新

## 页面验证点
- [ ] 页面加载显示至少 1-5 个常用点位卡片
- [ ] 搜索框输入后，卡片筛选更新
- [ ] 能拖拽卡片调整顺序
- [ ] 右键/菜单能正常打开
- [ ] 新增常用点位弹窗能打开
- [ ] 删除后卡片消失
