# PAGE_CONTEXT — DCS 上传日志（Upload-Log）

## 基本信息
- **页面名称**: 上传日志
- **路由**: `#/upload-log`
- **模块**: dcs
- **页面 ID**: dcs-upload-log

## 场景描述
DCS 数据上传日志查询页面。用户可查看采集设备的上报日志（成功/失败/异常），支持按时间、设备、状态筛选，用于故障排查和数据质量监测。

## 页面结构概览

| 区域 | 功能 | 组件类型 |
|-----|------|---------|
| **顶部工具栏** | 时间选择、设备筛选、状态筛选、搜索、导出 | DatePicker + Select + Input + Button |
| **统计卡片** | 上传总数、成功、失败、异常数量 | Statistic Card |
| **表格区** | 日志列表（时间、设备、点位、状态、错误信息） | Table |
| **分页** | 翻页、每页条数 | Pagination |

## 主要交互元素

### 时间和筛选区
- 时间范围选择：DatePicker（开始时间 - 结束时间）
  - 快捷选项：今天、最近 7 天、最近 30 天
- 设备筛选：多选下拉（允许选择多个设备）
- 状态筛选：多选下拉
  - 全部
  - 成功
  - 失败
  - 超时
  - 异常
- 搜索框：按错误信息/点位 ID 搜索
- 查询按钮
- 重置按钮

### 统计卡片
四个卡片显示当前筛选条件下的统计：
| 卡片 | 指标 | 颜色 |
|-----|------|------|
| 总数 | 本周期上传总数 | 蓝色 |
| 成功 | 成功条数 | 绿色 |
| 失败 | 失败条数 | 红色 |
| 异常 | 异常/超时条数 | 橙色 |

### 操作按钮
- 导出日志：导出当前筛选结果为 Excel/CSV
- 刷新：重新加载日志数据
- 清空日志：删除指定时间范围的历史日志（需双重确认）

### 表格列定义
| 列 | 数据类型 | 说明 |
|----|--------|------|
| 时间 | DateTime | 上传时间戳 (YYYY-MM-DD HH:mm:ss) |
| 设备 | String | 采集设备名/ID |
| 点位 | String | 关联的点位 ID |
| 点位名称 | String | 点位中文名 |
| 数据值 | String | 本次采集到的数据值 |
| 状态 | Badge | 成功(绿)/失败(红)/超时(黄)/异常(橙) |
| 错误信息 | String | 失败原因简述，成功则为 "-" |
| 重试次数 | Number | 上传重试次数 |
| 操作 | - | 详情 / 重试 |

### 操作列按钮
- 详情：查看完整日志信息（弹窗）
  - 完整错误堆栈
  - 重试历史
  - 数据预处理日志
- 重试：手动重新上传该条数据（仅失败状态可用）

### 日志详情弹窗
- 基本信息：时间、设备、点位、数据值
- 状态和错误信息
- 完整错误堆栈
- 重试历史列表
- 关闭按钮

## 已知复杂性
- 日志数据量可能很大（>10000 条/天），需分页加载
- 统计卡片需实时更新
- 日期选择需支持快捷范围（如"最近 7 天"）
- 状态颜色编码需保持一致性
- 重试功能需要权限控制
