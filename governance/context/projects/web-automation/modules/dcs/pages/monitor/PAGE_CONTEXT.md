# PAGE_CONTEXT — DCS 关键参数监控（Monitor）

## 基本信息
- **页面名称**: 关键参数监控
- **路由**: `#/monitor`
- **模块**: dcs
- **页面 ID**: dcs-monitor

## 场景描述
DCS 系统实时监控关键工业参数。用户可查看设备运行状态、温度/压力等关键指标实时曲线，支持告警配置与实时推送。

## 页面结构概览

| 区域 | 功能 | 组件类型 |
|-----|------|---------|
| **顶部工具栏** | 搜索、筛选、新增、刷新 | Input + Button |
| **主内容区** | 参数卡片网格 / 实时曲线图表 | Card / Chart |
| **侧栏** | 告警配置、参数详情 | Panel |
| **底部分页** | 翻页、每页条数控制 | Pagination |

## 主要交互元素

### 搜索/筛选区
- 搜索框：按参数名/设备 ID 搜索
- 筛选按钮：按状态（正常/告警/离线）筛选
- 时间选择器：选择时间段查看历史数据
- 重置按钮：恢复默认条件

### 操作按钮
- 新增参数：打开表单弹窗
- 刷新：重新加载实时数据
- 导出：导出参数数据为 CSV

### 数据展示
- 参数卡片：显示参数名、当前值、单位、状态（颜色编码）
- 实时曲线：绘制参数 10 分钟/1 小时 历史曲线
- 表格视图（可选）：参数列表 + 最新值 + 最后更新时间

### 操作列
- 编辑：修改参数配置
- 删除：移除参数监控
- 详情：查看参数完整信息 & 告警规则
- 告警设置：配置告警阈值

## 已知复杂性
- Vue 3 组件库使用 Element Plus 标准组件
- 实时数据推送可能使用 WebSocket（需处理刷新延迟）
- 参数卡片由 CSS Grid 布局，个数不固定（响应式）

- [ ] 页面加载完成时，至少显示 1 个参数卡片
- [ ] 搜索框输入关键字后，参数列表更新（<2s）

- [ ] 实时曲线图表能正常渲染（无白屏）
