## 高风险路径

- 删除客户 → 关联合同校验 → 后端拒绝/允许 → 前端提示 → 表格刷新（需全链路覆盖）
- 新增客户 → 必填校验失败 → 修正 → 重新提交 → 成功（错误恢复路径）
- 客户等级变更 → 关联合同审批链更新（跨模块联动）
