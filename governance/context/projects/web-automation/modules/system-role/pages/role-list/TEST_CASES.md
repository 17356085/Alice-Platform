### 3. 自动化优先级建议

| 优先级 | 建议策略 | 覆盖模块 |
|---|---|---|
| **高 (P0)** | 100%自动化 | 核心CRUD，预置角色保护，权限控制，E2E Happy Path (BS-ROLE-001) |
| **中 (P1)** | 80%自动化，剩余由手动覆盖 | 搜索/排序/分页基础操作，编辑冲突，角色删除限制 |
| **低 (P2)** | 选择性自动化 | 边缘权限场景、超长字符、性能测试，建议作为手动探索测试 |
