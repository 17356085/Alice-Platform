## 高风险路径

- 权限分配 → PC端500+checkbox → 保存 → 权限缓存刷新 → 用户重新登录 → 权限生效（全链路）
- 删除角色 → 关联用户检查 → 强制取消分配 → 确认删除 → 权限缓存刷新
- 分配用户 → 双栏穿梭 → 保存 → 用户获得角色所有权限（间接提权风险）

## 建议覆盖策略

- **必测（P0）**：权限缓存刷新（RISK-ROLE-001/012）、删除角色保护（RISK-ROLE-002）、提权防护（RISK-ROLE-005）
- **优先自动化**：CRUD主流程+状态筛选（定位器重构后A级占比提升）
- **手工保留**：500+checkbox全选性能（RISK-ROLE-006）
