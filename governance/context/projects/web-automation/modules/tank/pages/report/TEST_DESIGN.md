# TEST_DESIGN — tank / report

## 基本信息
- 模块：储罐管理（tank）
- 页面：储罐日报表（report）
- 测试目标：验证日报表数据展示、储罐切换、趋势图交互和导出功能
- 需求来源：PAGE_CONTEXT.md + RISK_MODEL.md

## 测试范围
- 包含：页面加载、储罐选择、日期选择、统计卡片、趋势图切换、导出功能
- 不包含：日报表后端数据计算准确性
