# TEST_DESIGN — dcs / all-data

## 基本信息
- 模块：DCS 数据监控（dcs）
- 页面：全部点位（all-data）
- 测试目标：覆盖点位列表的搜索/筛选/分页/批量选择全流程
- 需求来源：PAGE_CONTEXT.md + RISK_MODEL.md + AllDataPage.py

## 测试范围
- 包含：页面加载、搜索/重置、类型筛选、分页导航、行选择/全选、表格数据验证
- 不包含：新增/编辑/删除（页面为只读查询页，测试已 skip）

## 数据策略
- 测试数据来源：使用系统已有的点位数据（DCS模块真实数据）
- 数据清理方式：只读页面，无需清理
- 前置准备：确保 admin 账号有完整权限

## 自动化建议
- 建议自动化：搜索/分页/表格操作 — 定位器稳定（Element Plus 标准类）
- 不建议自动化：大数据量性能测试、图表数据准确性
