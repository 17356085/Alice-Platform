# RISK_MODEL — DCS 点位配置（Point-Config）

## 风险矩阵

| ID | 风险 | 可能性 | 影响 | 等级 | 缓解 |
|----|------|:-----:|:---:|:----:|------|
| R-PT-01 | 告警规则类型切换导致上下限字段条件显示 | M | M | **中** | 选择告警规则后用 `is_visible` 校验字段出现/消失 |
| R-PT-02 | 点位 ID 格式校验（前缀/数字规则） | L | M | **低** | smoke 测试用系统自动生成 ID |
| R-PT-03 | 采集设备下拉需异步加载 | M | M | **中** | 点击 select 后等 dropdown 出现再选 |
| R-PT-04 | 批量导入模板下载/上传格式兼容 | L | L | **低** | skip 导入导出 E2E，单独测试 |
| R-PT-05 | 删除点位后被其他页面引用导致失败 | L | H | **中** | cleanup fixture 注册失败时 warn_only |

## 测试影响
- **条件表单**: 告警规则联动字段需单独测试
- **导入导出**: E2E 不覆盖，标记 manual
- **清理**: cleanup fixture 用 `warn_only=True`
