# PAGE_CONTEXT — DCS 全部点位（All-Data）

## 基本信息
- **页面名称**: 全部点位
- **路由**: `#/all-data`
- **模块**: dcs
- **页面 ID**: dcs-all-data

## 场景描述
DCS 点位库管理页面。显示系统中注册的全部数据点（传感器/执行器/计算点），支持点位搜索、批量操作、数据上报率查看。

## 页面结构概览

| 区域 | 功能 | 组件类型 |
|-----|------|---------|
| **顶部工具栏** | 搜索点位名/ID、按类型筛选、导入/导出 | Input + Select + Button |
| **表格区** | 全部点位列表（点位ID、名称、类型、单位、状态、上报率） | Table |
| **批量操作** | 选择多个点位后的删除/禁用/启用 | Checkbox + Button |
| **分页** | 每页 10/20/50 条，支持跳页 | Pagination |

## 主要交互元素

### 搜索/筛选区
- 搜索框：按点位 ID / 名称 模糊搜索
- 类型筛选：下拉选择（模拟量/开关量/计数）
- 状态筛选：正常 / 故障 / 禁用
- 高级筛选：按设备、按采集周期
- 重置按钮

### 操作按钮
- 新增点位：打开点位表单弹窗
- 导入：上传 Excel/CSV 批量导入点位
- 导出：导出当前筛选结果为 Excel
- 刷新：重新加载点位列表

### 表格列定义
| 列 | 数据类型 | 备注 |
|---|--------|------|
| Checkbox | - | 用于批量选择 |
| 点位 ID | String | 唯一标识 |
| 点位名称 | String | 中文名称 |
| 类型 | String (Enum) | 模拟量/开关量/计数 |
| 单位 | String | 温度℃、压力MPa 等 |
| 采集周期 | Number | 毫秒，如 1000 = 1s |
| 状态 | String (Badge) | 正常/故障/禁用（彩色图标） |
| 上报率 (%) | Number | 实时数据上报完整度 |
| 操作 | - | 编辑 / 删除 / 详情 |

### 操作列按钮
- 编辑：修改点位参数（弹窗）
- 删除：移除点位（需确认）
- 详情：查看点位历史数据 & 统计

### 批量操作区
- 勾选多个点位后显示：
  - 启用 / 禁用 / 删除（批量操作）
  - 已选择 X 个点位

## 已知复杂性
- 点位数量可能很大（>1000），需分页加载
- 上报率是实时计算，可能有延迟
- 状态图标颜色编码需正确识别

## 页面验证点
- [ ] 页面加载显示至少 1 个点位行
- [ ] 搜索框输入后，表格筛选更新（<2s）
- [ ] 类型筛选下拉能正常打开关闭
- [ ] 选中多个行，批量操作按钮出现
- [ ] 分页能正常翻页
- [ ] 导出功能能生成文件
