# PAGE_CONTEXT — DCS 点位配置（Point-Config）

## 基本信息
- **页面名称**: 点位配置
- **路由**: `#/point-config`
- **模块**: dcs
- **页面 ID**: dcs-point-config

## 场景描述
DCS 点位配置管理页面。用户可创建新点位、编辑参数（采集周期、告警规则、数据处理）、批量导入/导出，并查看点位与采集设备的绑定关系。

## 页面结构概览

| 区域 | 功能 | 组件类型 |
|-----|------|---------|
| **顶部工具栏** | 搜索、筛选、新增、导入、导出、删除 | Input + Select + Button |
| **表格区** | 点位配置列表（ID、名称、类型、设备绑定、采集周期） | Table |
| **批量操作** | 选择多个点位后的启用/禁用/删除 | Checkbox + Button |
| **分页** | 翻页、每页条数 | Pagination |
| **表单弹窗** | 新增/编辑点位详细配置 | Dialog + Form |

## 主要交互元素

### 搜索/筛选区
- 搜索框：按点位 ID / 名称搜索
- 设备筛选：选择关联的采集设备
- 类型筛选：模拟量/开关量/计数
- 状态筛选：启用/禁用
- 高级筛选：按告警规则、按采集周期

### 操作按钮
- 新增点位：打开新增表单弹窗
- 导入：上传 Excel/CSV 批量创建点位
- 导出：导出当前筛选结果
- 批量删除：删除选中的点位
- 刷新

### 表格列定义
| 列 | 数据类型 | 说明 |
|----|--------|------|
| Checkbox | - | 批量选择 |
| 点位 ID | String | 唯一标识，自动生成或手填 |
| 点位名称 | String | 中文名 |
| 类型 | String (Enum) | 模拟量/开关量/计数 |
| 单位 | String | ℃、MPa、t/h 等 |
| 设备ID | String | 关联的采集设备 |
| 采集周期(ms) | Number | 采样频率 |
| 告警规则 | String | 无/上限告警/下限告警/双限告警 |
| 启用状态 | Badge | 启用/禁用 |
| 操作 | - | 编辑 / 删除 / 详情 |

### 表单弹窗字段（新增/编辑）
- 点位 ID（新增时自动生成，编辑时只读）
- 点位名称（必填）
- 点位类型（下拉：模拟量/开关量/计数）
- 单位（可选，如 ℃）
- 采集设备（下拉，必填）
- 采集周期（输入框，毫秒）
- 小数位数（数字输入，默认 2）
- 告警规则类型（下拉）
  - 无
  - 上限告警
  - 下限告警
  - 双限告警
- 告警上限值（条件显示）
- 告警下限值（条件显示）
- 告警说明（文本域）
- 启用状态（开关）
- 提交 / 取消按钮

### 操作列按钮
- 编辑：打开编辑表单
- 删除：删除点位（需确认）
- 详情：查看点位完整配置与历史修改记录

## 已知复杂性
- 表单具有条件显示逻辑（告警规则类型影响上下限字段显示）
- 点位 ID 可能有格式规范（如前缀命名）
- 设备下拉需要动态加载已配置的采集设备
- 批量导入需要支持模板下载
