### 2. PageObject 拆分方案

建议的 Page 类及职责：

- **UploadLogPage** (`pages/dcs/upload_log_page.py`)
    - 职责：封装上传日志列表页的所有操作，包括：
        - 搜索框输入（`search_input`）
        - 时间范围选择器（`date_picker`）
        - 状态下拉选择器（`status_select`）
        - 搜索/重置按钮（`search_btn`, `reset_btn`）
        - 表格行点击（`click_log_detail_by_text`）
        - 分页组件（`next_page_btn`, `page_size_selector`）

- **UploadLogDetailDialog** (`pages/dcs/upload_log_detail_dialog.py`)
    - 职责：封装日志详情弹窗的所有操作，包括：
        - 等待弹窗可见（`wait_for_dialog_visible`）
        - 获取字段值（`get_log_field_value`）
        - 点击关闭按钮（`close_dialog_btn`）

### 3. 公共组件复用分析

- **可复用BasePage已有方法**：
    - `find_element_click`：点击搜索、重置、分页按钮。
    - `find_element_send_keys`：在搜索输入框中输入文本。
    - `get_table_cell_text`：获取表格中特定单元格的文本（如文件名、状态）。
    - `wait_for_element_visible`：等待弹窗、表格加载完毕。

- **需要扩展ElementPlusHelper**：
    - **El-Select 封装**：页面有“状态”下拉框。`ElementPlusHelper` 需要新增 `select_by_label(dropdown_locator, option_label)` 方法来处理 Element Plus 的下拉选择。
    - **El-Date-Picker 封装**：页面有“时间范围”选择。需要新增 `select_date_range(start_date, end_date)` 方法，处理日期面板的联动选择。

### 4. 等待策略建议

该页面特有的异步行为：

1. **表格加载 Loading**：
    - 每次点击搜索、重置或切换分页后，表格上方会出现 `<div class="el-loading-mask">`。
    - **策略**：在 `UploadLogPage.search()` 方法中，点击搜索按钮后，显式等待该 Loading 元素出现并消失（`wait_until_element_disappear`）。

2. **详情弹窗动画**：
    - 点击表格行后，弹窗不是立即显示，会有一个滑入动画。
    - **策略**：在 `UploadLogDetailDialog.wait_for_dialog_visible` 中，使用 `visibility_of_element_located` 并设置较长 timeout (5秒)。

3. **分页数量刷新**：
    - 切换 “每页条数” 后，表格总数据量会重新计算，分页条显示的文字会更新。
    - **策略**：切换后，等待表格原本的第一行数据被 StaleElement 异常触发，或等待“加载中”状态消失。
