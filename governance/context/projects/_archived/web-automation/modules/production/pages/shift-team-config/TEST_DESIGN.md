## 自动化建议
- **Fixture 策略**：function 级 shift_team_config_page fixture（复用 module 级 driver_setup）
- **等待策略**：导航后等待页面身份标记（input placeholder）+ wait_vue_stable
- **定位策略**：搜索区用 placeholder；按钮用 XPath 文本；弹窗字段用 label 定位
- **弹窗清理**：fixture teardown 中 JS 关闭残留弹窗 + Escape 键兜底
