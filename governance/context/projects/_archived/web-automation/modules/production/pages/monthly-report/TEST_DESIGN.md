# TEST_DESIGN — production / monthly-report

## 基本信息
- 模块：生产管理（production）
- 页面：生产月报表（monthly-report）
- 测试目标：覆盖月份切换、报表生成、表格展示、趋势/导出弹窗
- 需求来源：PAGE_CONTEXT.md + RISK_MODEL.md

## 测试范围
- **包含**：页面加载验证、月份导航、生成报表、趋势/导出弹窗、空数据
- **不包含**：打印（浏览器原生对话框）、统计数据准确性（后端验证）
