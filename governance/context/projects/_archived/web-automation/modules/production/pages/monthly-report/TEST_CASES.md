## 页面概述

生产月报表是**只读**页面，按月展示产品/原料/公辅工程/冷剂消耗四类数据。核心交互包括：
- 月份切换（← → 按钮）
- 生成报表（刷新当月数据）
- 趋势查看（弹窗）
- 导出/打印

页面采用 Element Plus 自定义月份导航器，无数据录入功能。

## 自动化优先级建议

| 测试点 | 建议自动化 | 理由 |
|--------|-----------|------|
| TC-MR-001, 002, 003, 004 | ✅ 已自动化 | 已有稳定脚本 |
| TC-MR-007, 008, 009, 010 | 🔄 应自动化 | 边界月份导航，核心流程，手工耗时 |
| TC-MR-011, 012, 015, 016 | 🔄 可选自动化 | 数据校验，可结合 API 断言 |
| TC-MR-014, 022, 023 | 🔄 应自动化 | 核心业务场景，弹窗交互稳定 |
| TC-MR-017, 020, 021 | 🔄 推荐自动化 | 异常场景，可 mock |
| TC-MR-018, 019 | ❌ 不建议自动化 | 依赖多账号/权限环境，手工执行更灵活 |
| TC-MR-024 | ❌ 不建议自动化 | 浏览器打印对话框不可自动处理 |

> **不建议自动化原因**：权限测试需要多账号切换环境，打印功能涉及系统对话框。
