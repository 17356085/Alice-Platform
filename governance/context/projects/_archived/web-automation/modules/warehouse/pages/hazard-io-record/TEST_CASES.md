## 自动化优先级建议

| 等级 | 建议                                                                              |
|------|-----------------------------------------------------------------------------------|
| P0   | **全部自动化**。覆盖核心业务流程、数据隔离、权限、基本加载。                     |
| P1   | **≥80% 自动化**。异常场景（网络断开、超时等）可考虑 mock 后自动化，否则手工测试。|
| P2   | **≥50% 自动化**。边界值、列排序等可根据团队资源逐步自动化。                     |
| ❌ 不建议 | **SQL 注入**（需专业安全扫描工具）；**跨浏览/设备兼容性测试**（应与功能测试分离）。|
