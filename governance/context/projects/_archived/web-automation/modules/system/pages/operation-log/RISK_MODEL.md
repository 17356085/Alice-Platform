## 高风险路径
- 多条件组合搜索→结果验证（高频审计路径）
- 清空→确认→数据删除（不可逆危险路径）

## 建议覆盖策略
- 必测：P0清空确认、权限控制、多条件搜索
- 优先自动化：页面加载、搜索、详情、分页
- 手工保留：大数据量性能、敏感字段脱敏验证
