## 高风险路径
- 日志级别筛选(ERROR)→结果验证（高频故障排查路径）
- 清空→确认→数据删除（不可逆危险路径）

## 建议覆盖策略
- 必测：P0清空确认、权限控制、日志级别筛选
- 优先自动化：页面加载、搜索筛选、分页
- 手工保留：大数据量性能、异常容错
