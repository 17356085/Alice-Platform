## 高风险路径
- 搜索→详情查看（高频审计路径）
- 清空→确认→数据删除（不可逆危险路径）

## 建议覆盖策略
- 必测：P0清空二次确认、权限控制、搜索筛选
- 优先自动化：页面加载、搜索、详情查看、分页
- 手工保留：大数据量性能测试、导出功能验证
