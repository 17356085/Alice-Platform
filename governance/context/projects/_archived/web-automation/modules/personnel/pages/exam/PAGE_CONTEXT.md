# PAGE_CONTEXT — personnel / exam

## 基本信息
- 页面ID：personnel-exam
- 页面名称：考试管理
- 所属模块：人员管理（personnel）→ 培训管理
- 路由：`#/personnel/training/examArrange`
- 自动化代码：`page/personnel_page/ExamManagePage.py` + `script/personnel/test_exam_management.py`

## 页面职责
考试排程的完整管理。支持创建考试、关联试卷、指定考生、设置考试时间、阅卷评分。

## 核心元素（从代码推断）
- 搜索区：考试名称/编号输入 + 状态下拉 + 日期范围 + 搜索/重置
- 表格区：考试列表（编号/名称/关联试卷/开始时间/结束时间/状态/操作）
- 操作按钮：新增 / 编辑 / 删除 / 阅卷 / 查看成绩
- 弹窗：新增/编辑表单（考试名称/关联试卷/考试时间/考生列表/及格分数/考试时长）
- 分页区：分页器

## 关键交互
- 新增考试→弹窗→填写→关联试卷→指定考生→设置时间→确认→表格刷新
- 阅卷→查看考生答卷→主观题评分→提交成绩
- 考试时间到达后自动开始，结束后自动提交

## 权限与角色
- 可见角色：admin、培训管理员
- 可操作角色：admin、培训管理员（阅卷）
- 特殊限制：已结束的考试不可修改

## 特殊行为（待确认）
- 时间控制：考试开始/结束时间到达时自动触发状态变更
- 阅卷：主观题需人工评分，客观题自动评分
- 考生列表：支持按部门/岗位批量选择

## 依赖
- 接口：考试 CRUD + 阅卷评分接口
- 上下游页面：试卷管理（paper）、考试记录（exam-record）
