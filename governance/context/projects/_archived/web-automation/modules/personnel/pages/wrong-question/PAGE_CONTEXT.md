## 页面职责
- 展示用户在考试/练习中做错的题目集合
- 支持按课程/题型/时间等多维度筛选（推断，8 个 el-select + 8 个 el-form-item）
- 支持重新作答错题（推断，表单交互）
- 错题复习与巩固

## 关键交互
- 筛选 → 多条件组合筛选 → 查询 → 表格异步刷新
- 重新作答 → 点击 → 弹窗展示题目 → 作答 → 提交 → 即时判分 + 显示解析
- 查看解析 → 弹窗/展开 → 显示正确答案 + 解析 + 考点标注
- 移出 → 确认弹窗 → 题目从错题本移除
- 批量练习 → 选中多题 → 生成临时练习卷 → 开始答题

## 权限与角色
- 可见角色：所有登录用户（个人错题数据）
- 可操作角色：用户本人
- 特殊限制：仅显示当前用户的错题记录

## 特殊行为
- 异步加载：表格数据 + 8 个筛选下拉选项异步加载
- 动态渲染：题型/订正状态使用 el-tag 动态颜色
- 答题弹窗：组件化题目渲染（单选/多选/判断/填空不同组件）
- 即时判分：提交答案后即时显示对错，无需等待
- 个人数据隔离
- 空状态：无错题记录时显示 el-empty

## 依赖
- 接口：错题列表查询、多维度筛选、重新作答提交、移出错题本（待确认实际路径）
- 上下游页面：考试测评（my-exam，考试产生错题）、自主练习（practice，练习产生错题）、题库（question，题目来源）
