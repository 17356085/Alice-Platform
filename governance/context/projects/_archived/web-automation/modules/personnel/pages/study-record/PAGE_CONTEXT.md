# PAGE_CONTEXT — personnel / study-record

## 基本信息
- 页面ID：study-record
- 页面名称：学习记录
- 所属模块：人员管理（personnel）→ 培训管理
- 页面入口：左侧菜单 → 人员管理 → 培训管理 → 学习记录
- 路由：`#/personnel/training/studyRecord`
- 自动化代码：待开发

## 页面职责
- 展示所有学员的学习记录列表（⚠️ 非纯个人端，含学员名称列，admin 可见全员数据）
- 支持按学员名称/课程名称/完成状态多维度筛选
- 学习进度追踪（百分比 + 学习时长）

## 关键交互
- 筛选 → 输入学员名称/课程名称 + 选择完成状态 → 查询 → 表格异步刷新
- 点击课程名称 → 可能跳转课程详情（待确认）
- 分页切换 → 表格异步刷新

## 权限与角色
- 可见角色：admin（全员数据）、培训管理员（推断）
- 普通员工：可能仅可见自己的学习记录（待确认）

## 特殊行为
- 异步加载：表格数据 + 完成状态下拉异步加载
- 动态渲染：学习进度显示百分比，状态使用 el-tag
- 学习时长格式化：秒/分钟自适应单位
