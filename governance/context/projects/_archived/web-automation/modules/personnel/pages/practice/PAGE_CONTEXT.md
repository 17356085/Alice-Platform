# PAGE_CONTEXT — personnel / practice

## 基本信息
- 页面ID：practice
- 页面名称：自主练习
- 所属模块：人员管理（personnel）→ 培训管理
- 页面入口：左侧菜单 → 人员管理 → 培训管理 → 自主练习
- 路由：`#/personnel/training/practice`
- 自动化代码：待开发

## 页面职责
- 展示用户的自主练习记录列表（个人端页面）
- 支持按练习状态筛选（进行中/已完成）
- 支持对练习记录的三种操作：继续/开始练习、查看成绩、删除

| 未完成 | 继续练习 | 查看成绩 | 删除 |

### 分页区
- 分页器：el-pagination
- 当前数据：共 20 条，10 条/页，2 页

## 关键交互
- 筛选状态 → 下拉选择 → 表格异步刷新
- 开始练习 → 点击 → 跳转答题界面（新页面/弹窗，按顺序或随机出题）
- 继续练习 → 点击 → 回到上次练习进度
- 查看成绩 → 点击 → 弹窗展示成绩详情（正确/错误题目列表）
- 删除 → 行内删除按钮 → 确认弹窗 → 记录消失 + toast 提示

## 权限与角色
- 可见角色：所有登录用户（个人数据）
- 可操作角色：用户本人
- 数据隔离：仅显示当前用户练习记录

## 特殊行为
- 异步加载：表格数据 + 状态下拉异步加载
- 动态按钮：根据练习状态显示不同操作按钮（"开始练习" vs "继续练习"）
- 动态渲染：状态列使用 el-tag
- 正确率/得分：未完成练习显示 0.0% 和 0/N
- 空状态：无练习记录时显示 el-empty

## 依赖
- 接口：练习记录列表查询 / 删除 / 按状态筛选（待确认实际路径）
- 上下游页面：题库（question，题目来源）→ practice；practice → wrong-question（答题错误）
