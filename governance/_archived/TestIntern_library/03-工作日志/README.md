# 03-工作日志 — 日常记录

> 记录每日工作进展、每周总结、遇到的问题

---

## 日志类型

| 类型 | 频率 | 目录 | 模板 |
|------|------|------|------|
| 📅 日报 | 每天 | [日报/](日报/) | [_模板_日报.md](日报/_模板_日报.md) |
| 📊 周报 | 每周五 | [周报/](周报/) | [_模板_周报.md](周报/_模板_周报.md) |
| 🐛 问题记录 | 随时 | [问题记录/](问题记录/) | [_模板_问题记录.md](问题记录/_模板_问题记录.md) |

---

## 撰写规范

### 日报
- **时间**：每天下班前 10 分钟
- **内容**：今日完成的任务 + 遇到的问题 + 明日计划
- **长度**：简洁即可，5-10 行
- **命名**：`YYYY-MM/MMDD.md`，如 `2026-06/0609.md`

### 周报
- **时间**：每周五下午
- **内容**：本周完成汇总 + Bug统计 + 下周计划 + 风险/阻塞
- **长度**：半页到一页
- **命名**：`YYYY-WXX.md`，如 `2026-W24.md`

### 问题记录
- **时间**：遇到阻塞性问题时立即记录
- **内容**：问题描述 + 影响 + 解决方案 + 状态
- **长度**：不限
- **命名**：`YYYY-MM-DD_问题简述.md`

---

## 文件命名规范

```
日报：2026-06/0609.md           （按月份分目录）
周报：2026-W24.md               （ISO周号）
问题：2026-06-09_登录超时排查.md  （日期_简述）
```
