# 第1章 系统概述

## 1.1 文档说明

本文档为《鞍集（涂源）管理系统》微信小程序端的用户操作手册，基于系统原型图（V1.0.4）编写，用于指导用户进行各模块功能操作。

文档适用范围：

（1）公司内部归档及客户交付；

（2）运营人员培训参考；

（3）测试人员理解业务流程；

（4）后续系统维护参考。

## 1.2 系统登录

小程序支持两种登录方式，用户可根据实际情况选择。

### 1.2.1 微信一键登录

用户通过微信授权获取头像、昵称等基础信息，实现快速登录。登录成功后系统自动完善用户基础信息。

### 1.2.2 手机号登录

用户可通过输入手机号和密码完成登录认证，适用于未绑定微信或需要多设备登录的场景。

登录页面包含以下元素：

| 元素 | 说明 |
|------|------|
| 手机号输入框 | 输入已注册的手机号码 |
| 密码输入框 | 输入账户密码 |
| 登录按钮 | 提交登录信息进行认证 |
| 忘记密码 | 进入密码找回流程 |
| 注册账号 | 进入新用户注册流程 |
| 用户协议 | 勾选同意后方可登录 |

![图1-1 登录页面](prototype-screenshots/01-login.png)

## 1.3 多角色切换

若用户在系统中具有多个角色（例如同时具备普通员工与部门主管角色），可在【个人中心】页面顶部进行角色切换，切换后系统根据当前角色重新加载功能菜单。

![图1-2 个人中心页面（含角色切换）](prototype-screenshots/07-my-center.png)

## 1.4 用户角色与权限

![image-20260604175121069](./../../../docimg/image-20260604175121069.png)

系统设置四类用户角色，各角色对应不同的功能权限范围：

| 角色名称 | 权限范围 | 可见功能 |
|---------|---------|---------|
| 系统管理员 | 全部功能 | Web端所有菜单 + 小程序全部Tab |
| 普通员工 | 有限功能 | 小程序首页（部分模块）、审批、报警、我的 |
| 部门主管 | 部分管理 | 增加审批权限、本部门人员管理 |
| 安全员 | 安全相关 | 安全须知、报警处理、作业票审批 |

## 1.5 小程序主界面结构

小程序主界面底部导航栏划分为4个Tab页面，各Tab功能如下：

| Tab | 名称 | 主要功能 |
|-----|------|---------|
| Tab1 | 首页 | 工作待办、消息通知、储罐监控、设备状态、传感器、摄像头、在线学习、考试测评、错题集、自主练习、化验录入、环保危废、备品备件、销售录入 |
| Tab2 | 审批 | 审批中心（待审批/已审批/已驳回） |
| Tab3 | 报警 | 报警消息（全部/紧急/一般/已处理） |
| Tab4 | 我的 | 角色切换、我的申请、学习档案、消息设置、个人档案、使用帮助、关于系统、退出登录 |

![图1-3 首页主界面结构](prototype-screenshots/02-home.png)

![image-20260604175154203](./../../../docimg/image-20260604175154203.png)

（2）**功能卡片区**：以网格布局展示15个功能模块入口。

| 功能模块 | 所属区域 | 说明 |

| 储罐监控 | 生产监控 | 储罐实时运行状态监控 |

| 传感器 | 生产监控 | 传感器实时数据查看 |
| 摄像头 | 生产监控 | 现场摄像头画面查看 |

（3）**底部导航栏**：首页、审批、报警、我的四个Tab。

![图2-1 首页总览](prototype-screenshots/02-home.png)

## 2.2 消息通知

消息通知模块位于首页功能卡片区，用于展示系统推送的各类消息。

![image-20260604175212924](./../../../docimg/image-20260604175212924.png)

### 2.2.1 消息分类Tab

消息通知页面顶部提供三个筛选Tab：

| Tab | 说明 |
|-----|------|
| 未读 | 仅展示未读消息，右侧显示未读数量 |
| 已读 | 仅展示已读消息 |
| 全部已读 | 一键将所有消息标记为已读 |

### 2.2.2 消息列表

消息列表展示各类型通知，每条消息包含以下信息：

- **消息标题**：通知的主题内容
- **消息摘要**：简要描述通知内容
- **推送时间**：距当前的时间间隔（如"10分钟前"、"1小时前"、"今天 09:00"）

消息类型包括：

| 消息类型 | 示例内容 |
|---------|---------|
| 系统维护通知 | 系统将于今晚22:00-24:00进行维护升级 |
| 设备报警提醒 | 液氨储罐T-101压力超过预警值 |
| 审批待处理 | 张三提交的入场申请需要您审批 |
| 培训提醒 | 新培训任务《安全生产规范》待完成 |
| 通知公告 | 关于开展年度安全检查的通知 |

进入单条消息可查看详细内容，系统自动将未读消息标记为已读状态。

![图2-2 消息通知页面](prototype-screenshots/03-notification.png)

## 2.3 储罐监控

储罐监控模块用于实时监控LNG储罐的运行状态。页面展示储罐列表视图，包含以下关键参数信息：

| 参数 | 说明 |
|------|------|
| 储罐名称 | 储罐的设备编号或命名 |
| 存储客户 | 当前储存的客户名称 |
| 液位 | 当前液位高度（m） |
| 温度 | 当前罐内温度（℃） |
| 压力 | 当前罐内压力（MPa） |
| 运行状态 | 正常/异常/维修等状态 |

在储罐列表中选择单个储罐，进入储罐详情页面，展示以下内容：

（1）**基本信息**：储罐编号、容积、材质、工作压力、设计温度等静态参数；

（2）**实时参数**：当前液位、温度、压力、密度等动态数据；

（3）**传感器数据**：各传感器的实时检测值与状态；

（4）**状态信息**：储罐当前运行状态、上次维修时间等。

![图2-3 首页功能卡片（含储罐监控入口）](./../../../docimg/image-20260604175257907.png)

设备状态模块用于查看各类设备的在线/离线状态，帮助运维人员掌握设备运行情况。页面以列表形式展示设备信息，每条记录包含设备编号、设备名称、设备类型、运行状态（在线/离线/故障）等字段。

## 2.6 化验录入

化验录入模块用于录入化验室的分析检测数据，支持以下两类数据录入：

![image-20260604175458019](./../../../docimg/image-20260604175458019.png)

### 2.6.1 气体分析

气体分析数据录入页面用于记录气体成分的检测结果。页面包含检测日期、样品编号、气体成分名称、检测值、标准范围、检测结果（合格/不合格）等字段。录入完成后提交至系统，系统自动判定检测结果是否符合标准。

水质分析数据录入页面用于记录水质检测数据。页面包含检测日期、取样点、检测项目名称（如pH值、电导率、浊度、硬度、氯离子等）、检测值、标准范围、检测结果（合格/不合格）等字段。

# 第3章 人员培训

人员培训模块位于首页功能卡片区，包含在线学习、考试测评、错题集、自主练习四个子功能。首页待办统计区显示"待考试"数量，用于提醒用户有待完成的考试任务。



## 3.1 在线学习

在线学习模块提供课程学习和学习进度跟踪功能，用于帮助用户完成规定的培训课程。

![image-20260604175901720](./../../../docimg/image-20260604175901720.png)

### 3.1.1 课程列表

进入在线学习页面后，系统展示可用课程列表。每条课程记录包含以下字段：

| 字段 | 说明 |
|------|------|
| 课程封面 | 课程的封面图片 |
| 课程名称 | 课程的标题名称 |
| 课程类型 | 视频课程 / 文档课程 |
| 学习状态 | 未开始 / 学习中 / 已完成 |
| 更新时间 | 课程最后更新时间 |

课程列表支持按课程类型、学习状态进行筛选，便于用户快速定位目标课程。

### 3.1.2 视频学习

在课程列表中选择视频类型的课程，进入视频学习详情页面。页面包含以下内容：

（1）**视频播放器**：支持播放、暂停、进度拖动、全屏播放；

（2）**课程信息**：课程名称、讲师、课程时长；

（3）**学习进度**：显示当前完成百分比。

视频播放完成后，系统自动将学习状态更新为"已完成"，并记录学习时长。

![image-20260604175924584](./../../../docimg/image-20260604175924584.png)

### 3.1.3 文档学习

在课程列表中选择文档类型的课程，进入文档学习详情页面。页面以文档浏览形式展示课程内容，支持翻页浏览。完成全部内容浏览后，系统将学习状态更新为"已完成"。

![image-20260604175943090](./../../../docimg/image-20260604175943090.png)

## 3.2 考试测评

考试测评模块用于对用户的学习效果进行考核与评价，支持在线答题和成绩查看。

![image-20260604175958604](./../../../docimg/image-20260604175958604.png)

### 3.2.1 考试列表

![image-20260604180110382](./../../../docimg/image-20260604180110382.png)

进入考试测评页面后，系统展示可参加的考试列表。每条考试记录包含以下字段：

| 字段 | 说明 |
|------|------|
| 试卷名称 | 考试的试卷标题 |
| 考试时长 | 答题时间限制（分钟） |
| 总分 | 试卷满分值 |
| 及格分 | 通过考试的最低分数 |
| 考试状态 | 未开始 / 考试中 / 已完成 |
| 关联课程 | 考试所关联的培训课程名称 |

### 3.2.2 答题流程

![image-20260604180015402](./../../../docimg/image-20260604180015402.png)

在考试列表中选择待参加的考试，进入答题页面。答题页面包含以下内容：

（1）**题目区域**：展示当前题目内容、题目类型（单选/多选/判断）、分值；

（2）**答题区域**：提供选项供用户作答；

（3）**进度指示**：显示当前题号及总题数（如"3/25"）；

（4）**倒计时**：显示剩余答题时间；

（5）**交卷按钮**：用户完成全部题目后提交试卷。

考试过程中系统自动记录答题时间。达到考试时长限制后，系统自动提交试卷。



### 3.2.3 成绩查看

考试提交后，系统自动批改客观题并计算成绩。用户可查看考试成绩、各题目得分、正确答案及解析。

![image-20260604180126228](./../../../docimg/image-20260604180126228.png)

## 3.3 错题集

![image-20260604180141345](./../../../docimg/image-20260604180141345.png)

错题集模块自动收集用户在历次考试中答错的题目，按科目或课程分类展示，用于针对性复习。

### 3.3.1 错题列表

进入错题集页面后，系统展示错题列表。每条记录包含以下字段：

| 字段 | 说明 |
|------|------|

| 所属科目 | 题目所属的科目或课程 |
| 错误次数 | 该题被答错的累计次数 |
| 用户答案 | 用户提交的错误答案 |
| 正确答案 | 该题的标准答案 |
| 答题时间 | 最近一次答错的时间 |

### 3.3.2 错题复习

在错题列表中选择单条记录，可查看题目详情、正确答案及解析。系统支持按科目筛选错题，便于用户集中复习某一领域的薄弱环节。

## 3.4 自主练习

自主练习模块允许用户自行选择题目进行练习，练习成绩不计入正式考试记录，用于日常知识巩固。

| 错题试卷 | 系统自动从用户错题集中抽取题目生成练习试卷，用于针对性强化练习 |

![image-20260604180240222](./../../../docimg/image-20260604180240222.png)

![image-20260604180255439](./../../../docimg/image-20260604180255439.png)

### 3.4.3 练习历史

# 第4章 审批中心

审批中心为底部Tab2页面，用于处理各类审批业务，包括设备维保、领用申请、入场申请、离场申请、安全作业、培训申请等多种类型。

进入审批中心页面，顶部展示审批统计卡片，包含三类数据：

|---------|------|

## 4.2 审批类型筛选

审批中心提供搜索和类型筛选功能：

### 4.2.1 搜索功能

页面顶部提供搜索框，支持按申请编号或申请人姓名进行模糊搜索，输入关键词后系统实时筛选匹配的审批记录。

### 4.2.2 类型筛选Tab

默认显示"全部类型"，支持下拉选择以下审批类型：

| 类型 | 说明 |

| 安全作业 | 动火、高处等安全作业申请 |
| 培训申请 | 特种作业人员培训申请 |

选择审批类型后，列表自动过滤显示该类型的审批记录。

![image-20260604180400833](./../../../docimg/image-20260604180400833.png)

## 4.3 审批列表

审批列表以卡片形式展示每条审批记录，每条记录包含以下信息：

| 字段 | 说明 |
|------|------|
| 申请编号 | 审批单唯一编号（如 SB-2026-0038） |
| 审批状态 | 待审批 / 已审批 / 已驳回 |
| 申请标题 | 审批事项描述（如"液氨储罐压力传感器更换"） |

| 所属部门 | 申请人所在部门 |
| 提交时间 | 审批申请的提交时间 |
| 紧急程度 | 普通 / 重要 / 紧急 |
| 当前节点 | 当前审批所处环节（如"厂长审批"、"部门主管审批"、"安全主管审批"） |
| 操作按钮 | 审批 / 驳回 |

## 4.4 审批操作

在【待审批】列表中，每条审批记录下方提供两个操作按钮：

### 4.4.1 审批通过

审批人确认审批申请内容无误后，执行【审批】操作，系统将审批状态更新为"已审批"，审批流程流转至下一节点或完成。

### 4.4.2 审批驳回

审批人判定审批申请不符合要求时，执行【驳回】操作，系统将审批状态更新为"已驳回"，并将驳回结果通知申请人。

### 4.4.3 审批详情

进入审批记录卡片可查看审批申请的完整信息，包括申请内容、申请人信息、附件材料、审批流程记录等。
