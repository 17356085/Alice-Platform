### 4.2 修改 ArtifactStore 写入路径

文件: `aitest/platform/artifacts.py`

```python
class ArtifactStore:
    def _resolve_path(self, category: str, *parts) -> Path:
        """优先 .tlo/，fallback governance/context/projects/"""
        tlo = self._get_tlo_dir()
        if tlo:
            return tlo / category / Path(*parts)
        return self._legacy_dir / category / Path(*parts)
    
    def _get_tlo_dir(self) -> Optional[Path]:
        root = get_test_project_root()
        if root:
            tlo = root / ".tlo"
            tlo.mkdir(parents=True, exist_ok=True)
            return tlo
        return None
```

### 4.3 修改 paths.py 路径解析

```python
def get_context_modules(project_id=None) -> Path:
    tlo = _get_tlo_dir(project_id)
    if tlo:
        return tlo / "knowledge" / "modules"
    return _legacy_modules_dir(project_id)

def get_sop_status_dir(project_id=None) -> Path:
    tlo = _get_tlo_dir(project_id)
    if tlo:
        d = tlo / "runtime" / "sop-status"
        d.mkdir(parents=True, exist_ok=True)
        return d
    return _legacy_sop_status_dir(project_id)

def _get_tlo_dir(project_id=None) -> Optional[Path]:
    root = get_test_project_root(project_id)
    if root and (root / ".tlo").exists():
        return root / ".tlo"
    # 如果 .tlo/ 不存在但项目根目录存在，创建它
    if root and root.exists():
        tlo = root / ".tlo"
        tlo.mkdir(parents=True, exist_ok=True)
        return tlo
    return None
```

## 八、兼容性保证

> **详细策略**: [ZJSN_COMPAT_GUARANTEE.md](ZJSN_COMPAT_GUARANTEE.md)

核心原则:

| | 策略 |
|---|------|
| **读取** | `.tlo/` → 旧路径 → 报错（三级 fallback） |
| **写入** | P1-P2 双写（`.tlo/` + 旧路径），P3 后只写 `.tlo/` |
| **旧文件** | 保留原位标记 deprecated，不删除不移走 |
| **回滚** | `git revert`，旧路径始终可用 |

## 九、风险与回滚

| 风险 | 概率 | 缓解 |
|------|:--:|------|
| 旧 Agent 仍读旧路径 | 中 | paths.py 保留旧路径 fallback 2 周 |
| `.tlo/` 创建权限不足 | 低 | fallback 到 platform 目录 |
| ZJSN CI pipeline 依赖旧路径 | 中 | 归档不删除，CI 可继续用旧路径 |
| Skill 变量展开失败 | 低 | 保留原始字符串作为 fallback |

回滚: `git revert` + 恢复归档目录到原位。
